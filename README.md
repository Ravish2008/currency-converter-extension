# 💱 Universal Currency Converter

A Chrome extension that automatically detects and converts
currencies on any webpage.

## ✨ Features

- Auto-detects currencies on any website
- Supports symbols (₹, $, €, £ etc.)
- Supports Indian format (Crore, Lakh)
- Supports split prices (Nike, Adidas style)
- Mini currency converter in popup
- Crypto converter (BTC, ETH, SOL etc.)
- Searchable currency dropdown with flags
- Works on SPAs like X.com

## 🌐 Supported Sites

- Amazon
- Flipkart
- Nike
- Adidas
- 99acres
- Smartprix
- X.com (Twitter)
- And any other website with prices!

## 🚀 Installation

1. Clone this repo
2. Open Chrome → `chrome://extensions`
3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select this folder

## 🛠️ Tech Stack

- Vanilla JavaScript
- Chrome Extension Manifest V3
- Currency API: cdn.jsdelivr.net/npm/@fawazahmed0/currency-api
- Crypto API: CoinGecko
- Flags: flagcdn.com
